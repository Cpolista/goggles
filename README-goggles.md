# Peru Academic Priority Goggle

Custom Brave Search Goggle to prioritize academic sources for thesis research in Peru.

## File

- `peru-academic-priority.goggle`

## What It Does

- Boosts Peruvian thesis repositories and university domains.
- Boosts major scientific publishers and academic platforms.

## Raw URL

Use this URL when registering the Goggle in Brave:

`https://raw.githubusercontent.com/Cpolista/goggles/main/peru-academic-priority.goggle`

## Register In Brave

1. Go to: `https://search.brave.com/goggles/create`
2. Paste the raw URL above.
3. Submit and validate.

## Test Link

After registration, test with:

`https://search.brave.com/goggles?goggles_id=https%3A%2F%2Fraw.githubusercontent.com%2FCpolista%2Fgoggles%2Fmain%2Fperu-academic-priority.goggle`

## Update Process

1. Edit `peru-academic-priority.goggle` in this repo.
2. Commit changes.
3. Re-submit the same raw URL in `https://search.brave.com/goggles/create` to refresh Brave cache.
